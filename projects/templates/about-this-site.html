Technology and Background

I occasionally get asked for more details about this website so here they are!

Codebase

This site is developed using Python with Django as the web framework. For frontend HTML, CSS and Bootstrap is used.

Here are some more details:

  • Hosted on Heroku
  • Media files are hosted on Cloudinary
  • Local development and deployment are handled entirely using a Windows 10 running laptop
  • Python v3.8 used for development
  • Microsoft's Visual Studio Code used as the IDE
